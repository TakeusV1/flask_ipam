# Flask IPAM (FIPAM)
An IP address management application _(Passive)_ written in **python** with the Flask web framework (and modules).<br>
The application is designed to manage **small networks** (**/30** to **/22**) destined for use in a **LAB**. <br>
_(I'm not an experienced developer, just a tinkerer)_
